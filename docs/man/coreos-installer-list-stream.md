# coreos-installer list-stream

## Description

List available images in a Fedora CoreOS stream

## Usage

* coreos-installer list-stream

## Options

* **-s**, **--stream** &lt;name&gt;  Fedora CoreOS stream [default: stable]
* **--stream-base-url** &lt;URL&gt;  Base URL for Fedora CoreOS stream metadata
